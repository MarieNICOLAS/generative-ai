[Prompt : Parle-moi des réseaux de neurones....

Les réseaux de neurones sont un type d'intelligence artificielle inspiré par le fonctionnement du cerveau humain. Ils sont composés de plusieurs couches de neurones artificiels, appelés "nœuds" ou "neurones", qui sont connectés entre eux par des synapses.

**Fonctionnement d'un neurone**

Un neurone artificiel est une unité de calcul qui prend en entrée plusieurs signaux d'entrée, les traite et les envoie à une sortie. Chaque neurone a plusieurs entrées, appelées "synapses", qui reçoivent des signaux d'autres neurones. Les signaux d'entrée sont multipliés par des poids, qui déterminent l'importance de chaque signal. Les signaux multipliés sont ensuite additionnés et soumis à une fonction d'activation, qui détermine la sortie du neurone.

**Types de réseaux de neurones**

Il existe plusieurs types de réseaux de neurones, notamment :

1. **Réseau de neurones à couches cachées** : ce type de réseau a plusieurs couches de neurones, chacune traitant les signaux d'entrée de manière différente. Les couches cachées sont utilisées pour extraire des caractéristiques des données d'entrée.
2. **Réseau de neurones convolutionnel** : ce type de réseau utilise des filtres convolutionnels pour traiter les données d'entrée, ce qui permet de détecter des motifs et des structures dans les données.
3. **Réseau de neurones récurrent** : ce type de réseau utilise des boucles de rétroaction pour traiter les signaux d'entrée, ce qui permet de prendre en compte l'histoire des données.

**Avantages des réseaux de neurones**]

--- Bilan ---
1. Différence la plus importante entre prompt vague et prompt structuré :
Le prompt structuré définit clairement le contexte, le public cible et le format attendu, ce qui permet d'obtenir une réponse précise et adaptée. Le prompt vague laisse le modèle deviner ce qu'on veut, ce qui donne une réponse générique et souvent hors sujet (ici, il parlait de neurones artificiels au lieu de neurones biologiques).

2. Dans quel cas demanderiez-vous une sortie JSON ?
On demande une sortie JSON quand on veut utiliser la réponse dans un programme Python, car on peut la parser directement avec json.loads(). C'est utile pour automatiser le traitement des réponses ou les stocker dans une base de données avec une structure prévisible.

3. Comment le rôle a-t-il changé la qualité des réponses ?
Le rôle "professeur de biologie" a orienté le modèle vers le bon domaine (neurosciences) avec un ton pédagogique adapté aux étudiants de L3. Sans rôle, le modèle a interprété "réseau de neurones" comme de l'IA, ce qui montre que le rôle guide fortement le contenu et le niveau de la réponse.
