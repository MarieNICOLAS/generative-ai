print(hello)

Erreur : NameError: name 'hello' is not defined

Explique l'erreur et donne le code corrigé en 3-4 lignes max.


L'erreur `NameError: name 'hello' is not defined` signifie que Python ne connaît pas la variable `hello` que vous essayez d'utiliser. Cela se produit car vous n'avez pas défini la variable `hello` avant de l'utiliser.

Pour résoudre ce problème, vous pouvez définir la variable `hello` avant de l'utiliser. Voici le code corrigé :

```python
hello = "Bonjour"  # Définition de la variable hello
print(hello)  # Affichage de la variable hello
```

Alternativement, si vous voulez afficher le mot "hello" exactement comme il est écrit, vous pouvez utiliser des guillemets doubles pour spécifier que c'est une chaîne de caractères :

```python
print("hello")  # Affichage du mot "hello"
