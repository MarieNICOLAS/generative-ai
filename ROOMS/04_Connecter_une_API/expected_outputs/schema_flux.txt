Questions a remplir :
- Ou se trouve la cle API ? (cote client ou cote serveur ?)
  Reponse : Cote serveur uniquement (dans le fichier .env lu par mini_api_fastapi.py via utils.py)

- Pourquoi est-ce une bonne pratique de garder la cle cote serveur ?
  Reponse : Pour la securite ! Si la cle etait cote client, n'importe qui pourrait la recuperer 
  (via le code source, les outils de developpement du navigateur, etc.) et l'utiliser a nos frais.
  En gardant la cle sur le serveur, seul le serveur peut faire des appels a l'API payante.
